
# HR Analytics Dashboard

## Project Overview

The HR Analytics Dashboard is an interactive Power BI project designed to help Human Resource professionals monitor workforce trends, understand employee attrition, and make data-driven decisions.

The dashboard transforms raw HR data into meaningful insights through interactive visualizations, KPI cards, and dynamic filtering, enabling users to quickly identify patterns related to employee turnover, demographics, and overtime.

## Business Problem

Employee attrition can significantly impact organizational performance, recruitment costs, and productivity.

This dashboard was developed to answer key business questions such as:

- Which department experiences the highest employee attrition?
- Which age group has the highest turnover?
- Does overtime contribute to employee attrition?
- What is the gender distribution of employees leaving the organization?
- What is the overall attrition rate?

- Interactive Slicers
  - Department
  - Gender
  - Job Role
  - Education Field


## Key Insights

- The organization maintains a workforce of **1,470 employees**.
- Overall employee attrition stands at **16.1%**.
- Some departments experience noticeably higher attrition than others.
- Overtime appears to be associated with higher employee turnover.
- Employee demographics reveal differences in attrition across age groups and gender.

## Getting Started

1. Clone the repository.
2. Open the `.pbix` file using Power BI Desktop.
3. Refresh the dataset if required.
4. Explore the dashboard using the available slicers.

## Business Recommendations

Based on the dashboard analysis, the following actions are recommended:

- Investigate departments with the highest employee attrition to identify underlying causes.
- Review overtime policies and workload distribution to reduce employee burnout.
- Develop targeted retention strategies for high-risk employee groups.
- Strengthen employee engagement initiatives through regular feedback and career development programs.
- Monitor attrition trends continuously to evaluate the effectiveness of HR interventions.
